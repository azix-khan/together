# together

Get Together with TOGETHER, an Full Stack Social Media App built from scratch.

• Using Flutter, BLoC, Firebase 

## Getting Started

### This app includes:

🔒 Authentication. ✅

⚙️ Menu Drawer. ✅ 

🥷🏽 User Profile. ✅

🤪 Profile Picture. ✅ 

📸 Post - Upload & Delete. ✅

❤️ Post - Likes. ✅

💬 Post - Comments. ✅ 

🤝 Follow System. ✅

🔎 Search Users. ✅

🎨 Themes - Light & Dark Mode. ✅

🌎 Deploy to Web. ✅
